## Setup

You will need:

- [Node.js](https://nodejs.org/en/) (latest LTS version)
- [pnpm](https://pnpm.io/)

Install dependencies with `pnpm install`.

## Scripts

- `pnpm run start` - Run the project in production
- `pnpm run dev` - Run the project in development (with hot reloading)
